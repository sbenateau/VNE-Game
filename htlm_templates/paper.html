A partir des données collectées avec les protocoles Vigie-Nature Ecole, vos élèves pourront émettre des hypothèses pour expliquer la répartition des êtres vivants suivis. Ces hypothèses n'ont de valeur que localement, pour les valider, il est nécessaire d'avoir plus de données. Avec Galaxy Papers, ils accéderont à l'ensemble des données issues de Vigie-Nature École et pourront ainsi verifier si leurs hypothèses se confirment.

Comment utiliser Galaxy Papers ?

Commenez par télécharger le matériel de jeu en suivant ce lien.

Il existe deux possiblités pour jouer :

  • Vous pouvez utiliser des cartes avec un code (sous forme de lettres) inscrit dessus. Il vous suffira alors de taper dans le formulaire, le code de toutes les cartes utilisées (attention à bien respecter les majuscules et minuscules).

  • Vous pouvez utiliser des cartes avec un QR-code inscrit dessus. Il vous suffira alors de photographier la carte avec cette application. Vous avez donc besoin d'une webcam.
